🚀 Quickstart

Mulai pakai Apollo AI API dalam hitungan menit. Kompatibel dengan format OpenAI.

1. Buat API Key
Pergi ke halaman API Keys dan klik "Generate Key". Salin key yang dihasilkan.
2. Chat Completions — cURL
# Kirim request ke Apollo API curl https://apollo-api.vercel.app/v1/chat/completions \ -H "Authorization: Bearer apollo-sk-xxxx-xxxx-xxxx" \ -H "Content-Type: application/json" \ -d '{ "model": "apollo-core", "messages": [ {"role": "user", "content": "Halo Apollo!"} ] }'
3. Python
# pip install openai from openai import OpenAI client = OpenAI( api_key="apollo-sk-xxxx-xxxx-xxxx", base_url="https://apollo-api.vercel.app/v1" ) response = client.chat.completions.create( model="apollo-core", messages=[{"role": "user", "content": "Halo Apollo!"}] ) print(response.choices[0].message.content)
4. JavaScript / Node.js
// npm install openai import OpenAI from 'openai'; const client = new OpenAI({ apiKey: 'apollo-sk-xxxx-xxxx-xxxx', baseURL: 'https://apollo-api.vercel.app/v1' }); const res = await client.chat.completions.create({ model: 'apollo-core', messages: [{ role: 'user', content: 'Halo Apollo!' }] }); console.log(res.choices[0].message.content);
